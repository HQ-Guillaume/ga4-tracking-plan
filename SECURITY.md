# Security Policy

## Reporting

Please report security or privacy issues by opening a private security advisory on GitHub when available.

For ordinary analytics-quality issues, use a regular GitHub issue.

## Data Handling

Do not include the following in issues, pull requests, examples, or tracking-plan files:

- API keys or authentication tokens
- private GTM container exports
- GA4 measurement IDs for private properties when not intended for publication
- personal data, customer data, policy numbers, claim details, emails, phone numbers, or addresses
- proprietary client tracking plans unless they are explicitly approved for public release

## Supported Versions

Only the latest release is supported.
